LLM Actions Cheatsheet

Author: Tim Fraser, Systems Engineering
tmf77@cornell.edu
📝

SUMMARIZE

Condensing and distilling large volumes of text while preserving core intent.

Examples: Meeting minutes, TL;DRs, executive abstracts.

🔍

EXTRACT

Isolating and retrieving specific data points from unstructured "haystacks."

Examples: Pulling dates from emails, entity recognition, data scraping.

✍️

EDIT

Modifying existing content to improve quality, fix errors, or meet new constraints.

Examples: Grammar correction, rewriting for clarity, updating stale documentation.

🔄

REFORMAT

Transforming the structure, language, or tone without changing the core meaning.

Examples: Translation, JSON to CSV, jargon-to-layman conversion.

💡

GENERATE

Creating new content, concepts, or raw ideas from a prompt or seed idea.

Examples: Ideation, creative writing, drafting emails from scratch.

🎭

SIMULATE

Modeling a persona, system, or scenario to predict or practice reactions.

Examples: Roleplaying difficult conversations, persona testing, scenario modeling.

🏷️

CLASSIFY

Categorizing and labeling input into a specific taxonomy or bucket.

Examples: Sentiment analysis, intent detection, tagging support tickets.

✅

EVALUATE

Judging and scoring input against a rubric, logic gates, or specific criteria.

Examples: Grading essays, checking code for security flaws, compliance auditing.
