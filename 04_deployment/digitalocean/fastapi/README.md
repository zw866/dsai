# FastAPI Example

A minimal FastAPI REST API application for DigitalOcean App Platform deployment.

## What This App Does

This is a simple FastAPI application that provides several REST endpoints:
- `GET /` - Root endpoint with API information
- `GET /echo` - Echo back a message parameter
- `POST /sum` - Calculate the sum of two numbers
- `GET /docs` - Interactive API documentation (auto-generated by FastAPI)

## File Structure

- **`app.py`** - Main FastAPI application with route definitions
- **`requirements.txt`** - Python dependencies (fastapi, uvicorn)
- **`Dockerfile`** - Container configuration using Python 3.12-slim base image
- **`testme.sh`** - Local testing script

## How to Test

Run the test script to start the API locally:

```bash
./testme.sh
```

Or manually:

```bash
cd 04_deployment/digitalocean/fastapi
python -m uvicorn app:app --host 0.0.0.0 --port 8000
```

Once running, visit:
- `http://localhost:8000/` - Root endpoint
- `http://localhost:8000/docs` - Interactive API documentation
- `http://localhost:8000/echo?msg=hello` - Test echo endpoint
