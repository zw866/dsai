# Skill: source quality (disasters)

- Prefer **official** sources when **web_search** / preflight returns them: **federal / state / local emergency management**, **utility outage maps**, **NWS** / met agencies, **verified** org accounts.
- **`## References`** is the truth table for URLs: if a link is not in **`Retrieved URLs for References`**, it does not go in the brief.
- Treat **social posts and unverified blogs** as weak evidence; note that in **Key points** and keep **Confidence** low or medium.
- If results **conflict**, say so in **Key points** and lower **Confidence**—do not pick a “winner” without saying it is disputed.
- Snippets are not full articles; **paraphrase** in Key points and put the **canonical URL** in **References**.
