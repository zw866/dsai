# Skill: evidence brief shape (disaster context)

Use this shape for every answer unless the user asks for a different format.

1. **Question restatement** — Incident + geography + time window (if any).
2. **Key points** — At most three bullets; **no invented numbers or quotes**; align with retrieved sources only.
3. **References** — Numbered list of **real URLs** copied from **`### Retrieved URLs for References`** in preflight or `web_search` output; if none, the single “no URLs retrieved” line (see `references_section.md`).
4. **Confidence** — `low` / `medium` / `high` plus one sentence tied to whether you had live URLs.

Then output `END_BRIEF` alone on the last line when finished.
